# March 23, 2026 - AI Model News

## Sora 2 Video API Reaches Full Developer Access

OpenAI's Video API, powered by Sora 2, is now available to all developers following the March 13, 2026 public launch. The API represents a major milestone in programmatic video generation accessibility.

### Key Developments:

**API Capabilities:**
- **Video continuation**: Generate sequences that build on existing footage
- **Custom characters**: Maintain character consistency across clips  
- **Batch generation**: Process multiple video requests efficiently
- **Up to 20 seconds**: Extended generation length from previous 10-second limit
- **1080p resolution**: High-quality output standard

**Platform Integration:**
- **ChatGPT integration**: Sora 2 is being integrated directly into ChatGPT interface
- **Developer tools**: Full REST API with comprehensive documentation
- **Enterprise features**: Custom model fine-tuning available for select partners

**Market Impact:**
The API launch addresses a key developer pain point with video generation models. Previous limited access to Sora 2 API created development bottlenecks, with many teams building workarounds using multiple smaller models.

Early adopter feedback highlights the batch generation and video continuation features as game-changers for content workflows requiring consistent visual storytelling.

### Competitive Landscape Update

With Sora 2 API now widely accessible, the programmatic video generation market is seeing increased developer adoption compared to competitors:

- **Runway Gen-4.5**: API available but limited batch capabilities
- **Kling 3.0**: Strong model performance but API still in private beta
- **Veo 3.1**: Google's model lacks comprehensive developer tooling
- **Seedance 2.0**: ByteDance API remains restricted outside China

The broad API availability could accelerate Sora 2's adoption in automated content creation workflows, particularly for social media, advertising, and educational content.
