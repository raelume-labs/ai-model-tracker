# AI Model News - March 7, 2026

## OpenAI Launches GPT-5.4: Native Computer Use Era Begins
**Released:** March 5, 2026

OpenAI has launched GPT-5.4, marking a significant milestone as the first frontier model with native computer use capabilities. Described as "our most capable and efficient frontier model for professional work," GPT-5.4 represents a major step toward autonomous AI agents.

**Key Features:**
- **Native Computer Use:** First model with built-in computer vision for UI interaction
- **Professional Focus:** Advanced document handling for Excel, Sheets, presentations
- **Three Variants:** Standard, Pro (high performance), and Thinking (reasoning-focused)
- **Financial Integration:** Dedicated plugins for Microsoft Excel and Google Sheets
- **Autonomous Capabilities:** Enhanced reasoning and coding for agentic workflows

**Performance:**
- Improved performance on OSWorld-Verified computer use benchmarks
- Combines elite coding ability (on par with GPT-5.3-Codex) with superior world knowledge
- Enhanced vision capabilities for better computer interface interaction

**Pricing:** $2.50 per million input tokens with no free tier
**Availability:** ChatGPT, API, and developer platforms

## Alibaba Continues Model Innovation with Qwen3.5 Medium Series
**Released:** February 26, 2026 (recent coverage)

Alibaba's Qwen3.5 Medium series is gaining attention for delivering Sonnet 4.5-level performance on local hardware, marking a significant advancement in open-source AI accessibility.

**Highlights:**
- **Four Model Variants:** Three available for commercial use
- **Local Deployment:** Successfully runs on consumer M4 chip hardware
- **Agentic Tool Calling:** Built-in capabilities for autonomous task execution
- **Industry Recognition:** Elon Musk highlighted "impressive intelligence density"

**Impact:** Demonstrates growing viability of high-performance open-source models competing with closed-source frontier systems.

- **Multi-Agent System:** Four AI agents working in collaboration
