## How This Works

This repo tracks publicly announced AI model releases with a focus on:
- Image generation models
- Video generation models
- 3D generation models
- Audio/music generation models
- World models (interactive 3D environments)

Each category includes a comparison table with model specs, pricing, and key features.

## Contributing

Found a new model release? Open an issue or PR with:

- Release date (or announcement date)
- Key features
